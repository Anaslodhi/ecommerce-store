# Task 002: Global Layout, Navbar & Footer

## Status
Not Started

## Priority
High

## Description
Build the global application layout (`app/layout.tsx`) that wraps all pages with a consistent Navbar and Footer. The Navbar must be responsive (hamburger menu on mobile) and include navigation links to Home, Products, Cart, Login/Signup. The Footer should include branding, quick links, and social icons. Import Google Fonts (Inter) and set up the base Tailwind theme with the project's color palette.

## Acceptance Criteria
- [ ] `app/layout.tsx` includes metadata (title, description) and wraps children with Navbar + Footer
- [ ] `components/Navbar.tsx` renders logo, nav links (Home, Products, Cart, Login), and a mobile hamburger menu
- [ ] Navbar highlights the active route
- [ ] `components/Footer.tsx` renders branding, quick links, and social media icons
- [ ] Google Font (Inter) imported and applied globally
- [ ] Tailwind CSS theme extended with project color palette in `app/globals.css` or `tailwind.config.ts`
- [ ] Layout is fully responsive on mobile, tablet, and desktop
- [ ] No TypeScript errors or ESLint warnings

## Time Estimation
3 hours
