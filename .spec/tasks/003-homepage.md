# Task 003: Homepage

## Status
Not Started

## Priority
High

## Description
Design and build a stunning, modern homepage that serves as the storefront for TechNest. The page must include a hero section with a compelling headline and CTA, a featured products grid showcasing 3–6 top products, a value proposition / "Why Shop With Us" section, and a newsletter signup or additional CTA section. All sections must have smooth animations and premium aesthetics.

## Acceptance Criteria
- [ ] `app/page.tsx` renders the full homepage
- [ ] **Hero Section**: Eye-catching background/gradient, headline, subtitle, and a "Shop Now" CTA button linking to `/products`
- [ ] **Featured Products Section**: Displays 3–6 `ProductCard` components in a responsive grid, sourced from static product data
- [ ] `components/ProductCard.tsx` shows product image, name, price, rating, and an "Add to Cart" or "View" button
- [ ] **Value Proposition Section**: 3–4 feature cards (e.g., Free Shipping, 24/7 Support, Secure Payments, Easy Returns) with icons
- [ ] **CTA / Newsletter Section**: A visually distinct section encouraging users to subscribe or explore products
- [ ] `data/products.ts` contains at least 6 product objects with id, name, price, category, image, description, rating
- [ ] `types/index.ts` defines the `Product` TypeScript interface
- [ ] All sections are responsive and visually polished
- [ ] Smooth scroll animations or entrance effects on sections
- [ ] No TypeScript errors

## Time Estimation
3 hours
